SALES ANALYSIS ML

Overview

This project analyzes retail sales data using EDA and basic ML models do identify patterns and predict sales performance.

What I did

-Performed data cleaning and preprocessing;
-Conducted EDA to identify sales trend;
-Analyzed sales performance by category, region, and customer segment;
-Created time-based features (Year, Month, Day of Week);
-Built and evaluated machine learning models to predict sales;
-Compared Linear Regression and Random Forest models.

Results

-Sales increased overall between 2015 and 2018;
-Sales tend to peak toward the end of the year;
-The Technology category generates the highest revenue;
-The Consumer segment contributes the largest share of total sales;
-Machine learning models showed moderate predictive performance due to limited available features.

Tools used

Python, Pandas, NumPy, Scikit-learn.
